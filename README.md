# http-server

http static server for golang.

### Download

- [http-server](https://github.com/pengng/http-server/releases/)

### Install

```bash
go get github.com/pengng/http-server
```

### Usage

```bash
usage: http-server [options] [path]

options:
  -port        [port]
  -dir         [directory path]
  -help        help
```
